# GREPPER-CLI

Search for code snippets from your terminal.

## HOW TO INSTALL

```bash
npm i grepper-cli -g
```

## HOW TO USE IT

```
grepper-cli

Usage : ggp <your-search-text>
    ggp javascript recursion example
    grepper-cli javascript recursion example
```
